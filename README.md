# AI_Content_Generator

## Project Overview

AI Content Generator is a Python-based application that automatically generates content based on user-provided topics.

It helps users generate blog posts and social media captions based on a selected topic and tone. The application also displays word count and character count for the generated content.

## Features

* Blog Generation

* Content Creation

* Topic-Based Writing

* AI-Powered Text Generation

## Technologies Used

1. Python

2. Streamlit

## Applications

* Blog Writing

* Content Marketing

* Social Media Posts

* Product Descriptions

## Conclusion

This project demonstrates how Artificial Intelligence can automate content creation and assist writers in generating ideas and text efficiently.
